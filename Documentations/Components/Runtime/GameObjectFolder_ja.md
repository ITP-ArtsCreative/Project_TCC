# GameObjectFolder

#### **Namespace**: Unity.Utility
---

## 概要:
`GameObjectFolder` は、Unity エディタ内でゲームオブジェクトの階層構造を整理するためのコンポーネントです。このコンポーネントは、階層内のゲームオブジェクトを「フォルダ」として扱い、整理やコメントの追加を容易にします。

## 機能と操作:
- **コメントの追加**: エディタ上でゲームオブジェクトに対するコメントを追加できます。
- **階層色のカスタマイズ**: エディタの階層ビューでの表示色をカスタマイズできます。
- **子オブジェクトの表示制御**: ゲームオブジェクトフォルダ内の子オブジェクトの表示/非表示を切り替えることができます。

## プロパティ
| Name | Description |
|------|-------------|
| `_comment` | ゲームオブジェクトに対するコメント。 |
| `_menuColor` | 階層ビューでの表示色。 |
| `_isVisible` | 子オブジェクトの表示状態。 |
| `ChildObjects` | 子オブジェクトのリスト。 |

## 使用例
- `GameObjectFolder` コンポーネントをゲームオブジェクトに追加し、そのゲームオブジェクトを階層の「フォルダ」として使用します。
- `_comment` プロパティを使用して、ゲームオブジェクトに関するメモや説明を追加します。
- `_menuColor` プロパティを使用して、階層ビューでのこのフォルダの表示色を変更します。
- `_isVisible` プロパティを使用して、このフォルダ内の子オブジェクトの表示/非表示を切り替えます。

---
## その他の注意事項
- このコンポーネントは Unity エディタの機能拡張であり、主にエディタ上での使用を目的としています。ランタイムでの動作には影響しません。
- `Reset` メソッドは、コンポーネントがゲームオブジェクトに追加されたときに自動的に呼び出され、初期設定を行います。
- `ChildObjects` リストは、このフォルダに含まれる子オブジェクトを追跡するために使用されますが、エディタスクリプトによって動的に管理されることが想定されています。

`GameObjectFolder` コンポーネントは、大規模なシーンの管理や、複雑な階層構造を持つプロジェクトにおいて、階層の整理と可視化を支援する便利なツールです。
